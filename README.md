# Microsoft 365 Tenant Setup & User Management Lab:
Microsoft 365 Project displaying savvy pertaining to management of emails and users.

Built and configured a Microsoft 365 tenant via Microsoft 365 Developer Program. Created users, security and Microsoft 365 groups. Managing licensing, configuration of password via resets. Simulating onboarding and off-boarding processes.

<h2>Environments and Technologies Used</h2>

- Microsoft 365

<h2>Operating Systems Used </h2>

- Mac</b> 

<h2>Stages</h2>

- Display Users
- Create User Groups
- Create Security Groups
- Reset User Password
- Offboarding
